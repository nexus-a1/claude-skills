# Error Handling (Implement)

## No work found

```
✗ No incomplete work found in $WORK_DIR/

Start new work with /create-requirements
Or provide a requirements file: /implement path/to/requirements.md
```

## Branch conflict

```
⚠ Branch feature/{identifier} has diverged from remote

Options:
[p] Pull and rebase
[f] Force continue (may lose remote changes)
[a] Abort
```

## Commit failed

```
⚠ Commit failed: {error}

Options:
[r] Retry
[s] Skip this chunk
[a] Abort
```
